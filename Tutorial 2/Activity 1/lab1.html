Lab 1: Building a Simple Web Page

Objective

This lab focuses on creating a simple website, using semantically appropriate HTML5.

Steps Completed

  • Created 5 web pages having content as a summary of lectures in this course.
  • Created pages with consistent layout look and feel.
  • Inserted an image with proper attributes.
  • Built a vertical navigation menu.
  • Included a unique and descriptive title tag for each page using title best practices.
  • No styles, css, JavaScript, or other advanced components to the site.
