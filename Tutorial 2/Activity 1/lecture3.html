Lecture 3: Cascading Style Sheets (CSS)

Introduction to CSS

CSS stands for Cascading Style Sheets, which is a W3C standard for describing the appearance of HTML elements. With CSS, we can assign font properties, colors, sizes, borders, background images, positioning and even animate elements on the page.CSS can be added directly to any HTML element (via the style attribute), within the element, or, most commonly, in a separate text file that contains only CSS.

Benefits of CSS

It provides improved control over formatting. It improves site maintainability and accessibility. It improves page-download speed. It improves output flexibility. It provides responsive design.

Summary

There are three ways to add style to your HTML document: Inline CSS, Internal CSS, and External CSS. Inline style is used within the html element and only affects the element it is defined within. Internal (embedded) style sheet can be used to give style to one HTML page. External style sheet are a sperate files containing css code which is very useful for changing style of entire website with only one file. Padding is used to give space around element's content boders. A CSS framework is a set of CSS classes or other software tools that make it easier to use and work with CSS.
